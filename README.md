# Midas Backend Transaction Processing System

A Spring Boot backend service that processes financial transactions using Apache Kafka, validates and persists data using Spring Data JPA with H2 SQL, integrates an external Incentive REST API, and exposes a balance query endpoint.

## Tech Stack

- Java 17
- Spring Boot
- Apache Kafka
- Spring Data JPA
- H2 Database
- REST APIs
- Maven

## Features

- Kafka-based asynchronous transaction ingestion
- Transaction validation and balance updates
- Relational entity modeling
- External Incentive API integration
- REST endpoint for querying user balances

## Context

Completed as part of the JPMorgan Chase Software Engineering Virtual Experience Program on Forage.
